### criando venv

```
python -m venv venv
```

### acessando venv
```
source venv/bin/activate
```

### instalando requirements
```
pip install -r requirements.txt
```

### baixando datos
```
wget -O dados.csv https://covid.ourworldindata.org/data/owid-covid-data.csv
```

### rodando
```
python main.py
